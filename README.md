# Excel Sales Dashboard

## Project Overview
This project is an interactive Excel Sales Dashboard created to analyze sales performance, revenue, customer behavior, and marketing efficiency.

## Dashboard Features
- KPI Cards
- Pivot Tables
- Pivot Charts
- Slicers
- Sparklines
- Conditional Formatting
- Interactive Filters

## Key Metrics
- Total Sales
- Net Revenue
- Expected Revenue
- Refund Amount
- Discount Amount
- Bank Commission
- Income Tax
- Customer Count

## Dataset
The dataset includes:
- Sales ID
- Order Date
- Payment Date
- Customer Type
- Gender
- City
- Course
- Category
- Payment Status
- Payment Method
- Marketing Channel
- Ad Spend
- Revenue Metrics

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Power Query
- Formulas
- Slicers
- Conditional Formatting
